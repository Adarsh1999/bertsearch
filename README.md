```bash
$ wget https://storage.googleapis.com/bert_models/2018_10_18/cased_L-12_H-768_A-12.zip
$ unzip cased_L-12_H-768_A-12.zip
```

2. Run Docker containers.

```bash
$ docker-compose up
```

3. Index documents.

4. Go to <http://127.0.0.1:5000>.
